📊 Algorithm Comparison

From 1968 to 2024 — how bankruptcy prediction evolved, and why our GA+PSO approach outperforms every traditional method

🏛️

Previously Used Models Traditional / Existed

These models were used by researchers and banks before us. Each was the best in its era — but they all have limitations with modern data.

1968

📐 Altman Z-Score

Formula-based
F1 Score~25–30%
Accuracy~72%
Recall~38%
Features Used5 only
Why was it used?
Simple formula: Z = 1.2X₁ + 1.4X₂ + 3.3X₃ + 0.6X₄ + 1.0X₅. Can be calculated manually — no computer needed. Banks widely used this from 1970s to 2000s.
❌ Limitation: Only 5 fixed features, rigid linear formula, built on 1968 US data — does not generalize to modern global companies
1980

📘 Logistic Regression (Plain)

Ohlson O-Score
F1 Score~29–32%
Accuracy~89%
Recall~31%
Features Used9 ratios
Why was it used?
Ohlson (1980) applied Logistic Regression to bankruptcy prediction. Gives probability output. Interpretable — can explain why a company was flagged as bankrupt.
❌ Limitation: No feature selection, default hyperparameters, cannot handle class imbalance, linear decision boundary only
2005

🔷 Support Vector Machine

Kernel-based Classifier
F1 Score~33–38%
Accuracy~93%
Recall~35%
Features UsedAll features
Why was it used?
Min & Lee (2005) proposed SVM for financial distress prediction. Non-linear RBF kernel can capture complex patterns beyond linear boundaries.
❌ Limitation: Performs poorly on class imbalance (30:1 ratio), kernel and C tuning is difficult, slow on large datasets
1990s

🌿 Single Decision Tree

Rule-based Classifier
F1 Score~28–34%
Accuracy~91%
Recall~30%
Features UsedAuto-selected
Why was it used?
Fully interpretable — gives exact rules like "Debt Ratio > 0.8 AND ROA < 0 → Bankrupt". Easy to explain to auditors and regulators.
❌ Limitation: Severely overfits — memorizes training data but fails on new data. Highly unstable — small changes in data completely change the tree structure
1992

🧠 Early Neural Network

Shallow MLP
F1 Score~35–40%
Accuracy~94%
Recall~38%
Features UsedAll features
Why was it used?
Tam & Kiang (1992) — shallow 1–2 layer networks. Can capture non-linear patterns. Outperformed traditional statistical methods in early tests.
❌ Limitation: Overfits on small datasets like ours (6K rows), complete black box — no explainability, hyperparameters are very hard to tune
